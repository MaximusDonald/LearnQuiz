** Flux principal du système **

Utilisateur → React (upload cours)
           → FastAPI (réception fichier)
           → Parser Service (extraction texte)
           → Gemini API (génération quiz/résumé)
           → PostgreSQL (stockage)
           → React (affichage quiz/résumé)
           → Utilisateur (répond au quiz)
           → FastAPI (évaluation réponses)
           → Gemini API (analyse erreurs + explication)
           → React (feedback pédagogique)
